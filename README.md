# Oraculo d20 (front-end)

Projeto frontend inicial para uma plataforma de RPG online com Mestre IA.

Rodar localmente:

```bash
npm install
npm run dev
```

Stack: Next.js (App Router), TypeScript, Tailwind CSS.

Páginas implementadas: Home, Dashboard, Criar campanha, Sala da campanha, Criar personagem.

Este repositório contém apenas o frontend e usa dados mock para demonstração.
